# Task
```
1. Buat komponen Navbar untuk menampilkan 2 halaman (Home dan Products) (Gunakan React Route untuk Link)
2. Home menampilkan profil Perusahaan yang berisi 1 gambar perusahaan dan keterangan mengenai perusahaan
3. Halaman Products menampilkan daftar product dengan card sebanyak 3 card dalam 1 baris
4. Gunakan Axios untuk mengambil data produk. (https://dummyjson.com/products)
5. Dalam card 1 product terdapat 2 tombol, pertama view dan kedua add to cart
6. Ketika view menampilkan detail barang berisi gambar, judul, dan deskripsi (dari API)
7. Ketika add to cart akan menambah barang di keranjang (gunakan context untuk menambah barang ke state di context)
8. ketika cart di klik maka munculkan daftar barang sesuai dengan state.
```
